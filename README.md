OptLeaf-Webapp
-------

OptLeafのwebクライアント。

ドメインは準備中

* Node.js v16.14.0
* npm 8.3.1
* react.js ^18.1.0




コマンドログ
-------

```
# React app 作成 with typescript
npx create-react-app optleaf-webapp --template typescript

# React 設定ファイル可視化
npm run eject

# scss 対応
npm install -s node sass

#storybook セットアップ
npx -p @storybook/cli sb init --use-npm
```
